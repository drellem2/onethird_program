01 The one-paragraph state

If you remember nothing else, remember this.

The program has proven both endpoints of a single spectral axis and all the machinery that reduces the whole conjecture to one implication. Nearly every quantity we track — λ_std, inversion count, squared displacement, interface thinness, entropy — is the same axis (“near-ordinal-sumness”) in different units. The balance constant δ is a separate axis (the counterexample condition). The entire remaining gap is the one bridge between them:

the wallL1b — frozen forces near-ordinal-sum
δ(P) < 1/3  ⟹  λ_std → 1   (equivalently  E[inv_e] = O(n/γ)  or  E[Σ disp²] = O(E[inv_e]))

Everything else is either proven, or reduces to this. It is hard because it must use that σ ranges over a real poset's linear extensions — it is false for abstract frozen distributions.

03 The regime map

Where a minimal counterexample must live, and why the data cannot see it. A conceptual map — not a scatter plot; δ is not even measured for most posets.

The vertical δ = 1/3 line is the counterexample threshold. A minimal counterexample must sit in the hatched top-left danger corner (frozen and near-ordinal-sum) — which is empirically empty: no frozen poset is reachable at all (δ bottoms out near 0.40). The ordinal-sum top edge is non-primitive, so excluded. The one anchored boundary point, tight3, is the conjecture-tight case (δ = 1/3 exactly, a just-balanced pair) — not a counterexample.

Correctness note — do not conflate δ and Δ

Three empirical facts must stay separate; merging them manufactures a false “imbalance ⟹ good mixing” law that contradicts a proven theorem:

(i) λ_std → 1 ⟺ thin interface — empirical, corr(1−λ_std, Δ₁)=+0.776. This is interface fatness Δ, a cut-geometry quantity — not the balance constant δ.   (ii) No frozen poset reachable; δ ≥ 0.40 at width ≥ 4, monotone in the (B)-ratio R = E[Σdisp²]/E[inv] — a δ-vs-R fact, silent on δ < 1/3.   (iii) frozen ⟹ λ_std → 1 — this is conjectural (L1b), untested in reach. There is no measured δ-vs-λ_std correlation; δ is not even a field in the dataset.

06 The single lemma to prove

Invariant form — no cut, no ledger, no width. Two faces; either one closes the wall via already-proven machinery.

open · the wallPoset-LE displacement anti-concentration

For every finite poset P with distinguished order e and δ(P) < 1/3, with σ uniform on the linear extensions L(P):

displacement face (B):   E[ Σₓ (pos_σ(x) − rank_e(x))² ]  =  O( E[ Σₓ |pos_σ(x) − rank_e(x)| ] )
inversion face (LIB):   E_σ[ inv_e(σ) ]  =  O( n / γ )

“A random linear extension of a real, 2/3-frozen poset stays close to its reference order — no heavy displacement tail / only linearly many inversions.” The two faces are logically independent (a single deep-crossing element separates them); either alone suffices.

forced: real-poset LE measure forced: δ < 1/3 forced: distinguished order e

Why it is hard — obstruction 4

Both faces are false for abstract frozen distributions (a two-atom law has every pair frozen yet Θ(n²) inversions). So the proof must use that σ ranges over a genuine poset's linear extensions — not just that pair marginals are frozen. This kills the marginal-only tools: slot-law log-concavity is numerically false, and FKG / XYZ are wrong-signed. The needed input is a real-poset (order-polytope / joint-law) anti-concentration — the same crux the Čech program also imports. The untried handle: weak-Bruhat convexity / Stanley absolute-position AF log-concavity forcing the slot probabilities to decay.

07 Attempt index

What has been tried and its verdict — so nothing is re-walked. Each of these would have been a one-line lookup during the sessions that produced them.

dead end

Bruhat-convexity: “prefix violations ⟹ O(n) inversions”

Collapses entirely to Diaconis–Graham. Prefix-violations = footrule ≍ inversions — one equivalence class up to a factor of 2. No easier target; convexity does no work at this level.

avoid this aim

balance → log e(P), aimed at δ directly

This is the Kahn–Saks / Kahn–Linial line, stuck at δ ≥ 0.276 for 30 years. A frozen counterexample sits in the open gap it can't exclude; the whole program was built to escape this route. Do not point entropy at δ.

untried · open

Entropy / order-polytope, aimed at the conditional inversion sum

Genuinely untried, doc-endorsed in category, not blocked by the dead slot-law. Deliverable: the joint→marginal bridge log e(P) → E[inv_e], using order-polytope geometry (AF / Stanley absolute-position log-concavity) to beat the two-atom counterexample. Young-lattice / “harmonic-area” proposals decode to this same route.

untried · convergent target

Convexity / order-polytope forbids the flat slot law

Convexity does dispense with the two-atom witness (Reverse-Cheeger §4.2: the support is order-polytope lattice points, not two atoms), and positivity e(P_m)>0 kills the exact zero-interior block-cross — but both are silent on the approximate flat tail, the real LIB-violator. Using weak-Bruhat convexity / connectedness or Stanley's absolute-position AF log-concavity (the true theorem, ≠ the dead relative-to-chain one) to force ρ_s<1 is unexplored — zero corpus hits. This is where the threads converge.

dead ≠ AF

slot-law log-concavity of e(Pₘ)

Numerically false (Neggers–Stanley object, 7,537 violations). But this is the relative-to-chain sequence — it does not touch Stanley's absolute single-element position log-concavity (a genuine AF theorem), which remains available.

untried · new object

Incomparability-graph conductance → δ / inversion sum

The “primitive ⟹ good mixer” form in λ_std is refuted (primitives reach λ≈0.96). But the incomparability graph's own expansion — a graph on n elements, distinct from the transport Laplacian — is never examined. Precise question: can freezing keep a dense incomparability cut low-leakage in a real poset? Same wall, small new object.

Where the threads converge

Convexity dispensing the two-atom case, an entropy-gradient discontinuity at δ = 1/3, and Brightwell's open question (can a poset sequence reach δ → 1/3?) are not three problems — they are three windows on one unproven statement: a real poset's uniform / connected / order-polytope structure forces the slot probabilities to decay (ρ_s < 1). Prove that, and the wall falls. Empirically no poset reaches δ ∈ (1/3, ≈0.354); the best known constructions plateau at ≈0.349 — the forbidden-band signature, still open at n → ∞.
